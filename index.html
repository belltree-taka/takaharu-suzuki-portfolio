Challenge

I'm actively incorporating new knowledge as well.

Project Name:

React Todo App

https://react-zen-do-app.netlify.app/

Key Points:

I challenged myself to implement a simple Todo app using React and Material UI. I focused on using Material UI components for the interface and managing the app's state with React. It was a good opportunity to practice these technologies and improve my development skills.

Description:

I took the time to thoroughly learn JavaScript syntax before diving into React. Through this project, I deepened my understanding of how the virtual DOM works based on React’s state. As you can see from the source code, the program still contains a lot of redundancy and requires refactoring.

However, I implemented features such as adding, deleting, and updating Todo items, as well as various warning messages. I also challenged myself with styling using Material UI. Through personal projects like this, I aim to not only apply what I’ve learned in school but also take on new challenges.
